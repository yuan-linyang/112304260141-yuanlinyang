# 实验 2 运行指南

## 📋 代码版本说明

本实验提供了两个版本的代码：

### 1. 基础版本（不推荐）
- 文件：`code/word2vec_sentiment_analysis.py`
- 特点：使用随机森林，删除所有停用词（包括否定词）
- 预期 AUC：0.85-0.90
- **不推荐使用**：会误判否定句的情感

### 2. 优化版本（推荐）⭐
- 文件：`code/word2vec_sentiment_analysis_v2.py`
- 特点：
  - ✅ 保留否定词（not, no, never 等）
  - ✅ 提取否定短语和情感短语
  - ✅ 使用加权平均句向量
  - ✅ 使用逻辑回归分类器
- 预期 AUC：**0.94-0.96**
- **强烈推荐使用此版本**

## ⚙️ 参数调优

如果 AUC 未达到 0.95，可以调整以下参数：

### 1. Word2Vec 参数
```python
num_features = 300    # 词向量维度（建议 200-400）
min_word_count = 10   # 最小词频（建议 5-20）
context = 10          # 上下文窗口（建议 8-12）
epochs = 10           # 训练轮数（建议 10-20）
```

### 2. 逻辑回归参数
```python
C = 1.0              # 正则化强度（建议 0.1-10.0）
max_iter = 1000      # 最大迭代次数（建议 500-2000）
```

### 3. 否定词权重
```python
weight = 1.5  # 否定词权重（建议 1.3-2.0）
```

## 🔍 常见问题

### Q1: 运行速度慢
**A:** Word2Vec 训练需要时间，首次运行约 10-15 分钟。后续可加载已保存的模型：
```python
model = Word2Vec.load("code/word2vec_model_optimized")
```

### Q2: 内存不足
**A:** 减少训练数据量或降低词向量维度：
```python
num_features = 200  # 降低维度
```

### Q3: AUC 低于 0.94
**A:** 尝试以下优化：
1. 增加 Word2Vec 训练轮数：`epochs=20`
2. 调整逻辑回归正则化：`C=0.5`
3. 使用更多未标注数据训练 Word2Vec

### Q4: 导入错误
**A:** 确保安装了所有依赖：
```powershell
pip install pandas numpy gensim scikit-learn beautifulsoup4 nltk
```
