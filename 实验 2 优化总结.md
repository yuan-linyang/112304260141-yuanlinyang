# ✅ 实验 2 代码优化完成！

## 📊 优化总结

### 核心改进

针对你的要求，我已完成以下优化：

#### 1. **保留否定词** ✅
- **问题**：传统方法删除 "not"、"no"、"never" 等否定词，导致 "not good" 被误判为负面
- **解决**：
  - 创建否定词列表（25 个常见否定词）
  - 预处理时保留所有否定词
  - 对否定词赋予 1.5 倍权重

#### 2. **使用短语模式** ✅
- **问题**：单个词无法捕捉否定结构
- **解决**：
  - 提取包含否定词的短语（如"not good"、"not bad"）
  - 提取情感短语（如"very good"、"highly recommend"）
  - 将短语作为额外特征加入训练

#### 3. **使用简单模型** ✅
- **问题**：复杂模型（如随机森林）容易过拟合
- **解决**：
  - 使用**逻辑回归**（Logistic Regression）
  - 参数优化：C=1.0, max_iter=1000, solver='lbfgs'
  - 类别权重：balanced（自动处理类别不平衡）

#### 4. **Word2Vec 优化** ✅
- Skip-gram 模型（sg=1）：对罕见词有更好的表示
- 降低最小词频：min_count=10（保留更多词汇）
- 增加上下文窗口：window=10
- 负采样：negative=5，提高训练效率

#### 5. **加权平均句向量** ✅
- 普通词权重：1.0
- 否定词权重：1.5
- 增强否定词的表达能力

## 🎯 关键优化点

### 1. 否定词保留
```python
NEGATION_WORDS = {'not', 'no', 'never', 'nothing', 'nowhere', 
                  'neither', 'nobody', 'none', 'nor', 
                  'cannot', 'can\'t', 'won\'t', 'wouldn\'t', 
                  'shouldn\'t', 'couldn\'t', 'didn\'t', 
                  'doesn\'t', 'don\'t', 'isn\'t', 'aren\'t', 
                  'wasn\'t', 'weren\'t', 'hasn\'t', 'haven\'t', 'hadn\'t'}

# 对否定词赋予更高权重
weight = 1.5 if word in NEGATION_WORDS else 1.0
```

### 2. 短语提取
```python
SENTIMENT_PHRASES = {'very bad', 'very good', 'too much', 'so great', 
                     'not good', 'not bad', 'not great', 'not boring',
                     'highly recommend', 'must see', 'waste of time',
                     'best movie', 'worst movie', 'really enjoyed'}

# 提取包含否定词的短语
```
